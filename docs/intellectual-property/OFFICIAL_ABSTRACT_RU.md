# РЕФЕРАТ
## Программный комплекс «Digital Audit System (DAS)» версия 0.5

**Область применения**: Автоматизация технического и нормативного аудита объектов недвижимости, промышленных предприятий и систем безопасности на соответствие действующему законодательству Республики Казахстан (СН РК, СП РК, Технические регламенты).

**Функциональное назначение**:
Программный комплекс обеспечивает цифровую трансформацию процессов аудита и включает:
1. **Интеллектуальный модуль парсинга**: Трансформация неструктурированной нормативной документации в базу атомарных технических требований с применением больших языковых моделей (LLM).
2. **Систему аддитивного хранения данных**: Технология, позволяющая сохранять экспертные корректировки и метаданные требований при обновлении исходных нормативных текстов.
3. **Мобильное приложение (Field App)**: Обеспечивает проведение полевых работ в режиме оффлайн с сохранением локальных данных в IndexedDB и последующей транзакционной синхронизацией.
4. **Аналитический модуль**: Автоматическое формирование реестра дефектов, планов корректирующих мероприятий (CAPA) и предварительной сметной стоимости (ROM Budget) в национальной валюте (Тенге).
5. **Генератор отчетности**: Автоматический выпуск пакета документов «Audit Pack» в форматах PDF/DOCX (Executive Summary, Technical Report, Photo Evidence).

**Технические характеристики**:
- Стек технологий: Next.js, TypeScript, Tailwind CSS, Prisma ORM.
- База данных: PostgreSQL / Supabase.
- Оффлайн-хранилище: Dexie.js (IndexedDB).
- Интеграция: OpenAI API (модели семейства GPT-4o-mini / GPT-4o).
